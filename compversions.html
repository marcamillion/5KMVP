Project overview

A web application that allows a designer to upload multiple versions of an image, then easily send a link to a client that doesn't require them to login. He can then leave comments and quickly vote up/down on each of the images. The system automagically fills in the correct client information, based on the URL that was visited.

Technology

  • HTML / CSS / jQuery
  • Ruby on Rails, PostgreSQL
  • Hosted on Heroku
  • Images stored on Amazon S3
